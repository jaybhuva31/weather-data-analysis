# 🌦 Weather Data Analysis Project

## 📌 Overview
This project focuses on analyzing a real-world weather dataset using Python. The goal is to extract meaningful insights from raw data and understand patterns in temperature, humidity, and rainfall.

---

## 🎯 Objectives
- Perform data cleaning and preprocessing  
- Analyze temperature and humidity trends  
- Identify rainfall patterns  
- Visualize data using charts and graphs  
- Generate insights for real-world decision making  

---

## 🛠 Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## 📂 Dataset
- Weather dataset (CSV format)  
- Contains features like temperature, humidity, rainfall, etc.

---

## 🔍 Project Steps

### 1. Data Cleaning
- Checked missing values  
- Removed null values using dropna()  

### 2. Feature Engineering
- Created AvgTemp (average of MinTemp & MaxTemp)  
- Created AvgHumidity (average of morning & afternoon humidity)  

### 3. Data Analysis
- Temperature analysis (min, max, average)  
- Humidity analysis  
- Rainfall frequency analysis  

### 4. Data Visualization
- Histogram (Temperature distribution)  
- Bar chart (Rain frequency)  
- Heatmap (Correlation analysis)  
- Boxplot (Humidity vs Rain)  
- Pie chart (Rain distribution)  

### 5. Extreme Weather Detection
- Identified hottest and coldest days using idxmax() and idxmin()  

---

## 📊 Key Insights
- High humidity is strongly associated with rainfall  
- Temperature varies significantly across days  
- Rainfall distribution shows imbalance between rainy and non-rainy days  

---

## 🚀 Conclusion
This project demonstrates how raw weather data can be transformed into meaningful insights using Python. It highlights the importance of data cleaning, visualization, and analysis in real-world scenarios.
